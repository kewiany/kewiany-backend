# 📝 kewiany-python

## 🛠 Tech stack

- [Python](https://python.org/) 3.7.7
- [Flask](https://flask.palletsprojects.com/en/1.1.x/) 1.1.2
- [Pytest](https://docs.pytest.org/en/latest/) 5.4.2

## 🧰 Installation

```sh
pip install -r requirements.txt
```

```sh
python -m venv env
```

```sh
source venv/bin/activate
```

```sh
source .bash
```

```sh
flask run
```

```sh
pytest
```
